Gerador de Senhas Seguras

O limite de caracteres é 99.

Sobre o projeto

O Gerador de Senhas Seguras é uma aplicação web desenvolvida em Python com Flask no backend e HTML, CSS e JavaScript no frontend. O objetivo deste projeto é permitir que os usuários possam gerar senhas seguras de forma fácil e personalizável.

Funcionalidades:

Personalização da Senha: Os usuários podem personalizar o comprimento da senha e escolher se desejam incluir letras maiúsculas, números e caracteres especiais.

Validação Dinâmica: O comprimento máximo da senha é ajustado dinamicamente com base no valor inserido pelo usuário para garantir que ele não exceda o limite permitido.

Geração de Senhas: Quando o usuário submete o formulário, uma requisição é enviada ao servidor Flask, que gera uma senha aleatória baseada nas preferências do usuário.

Exibição da Senha Gerada: A senha gerada é exibida na página para que o usuário possa copiá-la e usá-la conforme necessário.

Benefícios:

Segurança Aprimorada: Senhas geradas aleatoriamente são mais seguras do que senhas criadas manualmente, pois são menos previsíveis para ataques.

Facilidade de Uso: A interface intuitiva torna simples para os usuários personalizarem e gerarem suas senhas de forma rápida e conveniente.

Flexibilidade: A capacidade de personalizar o comprimento e os tipos de caracteres permite que os usuários adaptem as senhas às necessidades específicas de segurança de cada situação.
