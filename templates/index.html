Gerador de Senhas Seguras

O limite de caracteres é 60.

Sobre o projeto

O Gerador de Senhas Seguras é uma aplicação web desenvolvida em Python com Flask no backend, acompanhado de Javascript com HTML e CSS. O objetivo deste projeto é permitir que os usuários possam gerar senhas seguras de forma fácil e personalizável.

Funcionalidades:

  • Personalização da senha com comprimento e tipos de caracteres.
  • Validação dinâmica do comprimento máximo permitido.
  • Geração de senhas aleatórias pelo backend em Python.
  • Exibição direta da senha gerada ao usuário.

Benefícios:

  • Senhas mais seguras e difíceis de prever.
  • Interface simples e intuitiva.
  • Flexibilidade para diferentes necessidades de segurança.
